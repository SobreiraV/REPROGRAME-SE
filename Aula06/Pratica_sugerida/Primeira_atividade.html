Práticas Sugeridas da Aula 06

Atividade 01

Objetivo: Criar uma página contendo os vídeos desse módulo separados por semana, usando somente os recursos HTML que estudamos até o momento.

♦ 1ª Semana ♦

Aula 01 :: Como Funcionam os Sites

Nessa videoaula, eu mostro como funcionam os sites nos bastidores, que tipo de informação trafega entre o navegador do usuário e o servidor web, tudo de uma maneira bem simplificada e didática.

Prof. Ricardo Maroquio


Aula 02 :: A Estrutura Básica de Arquivos HTML e CSS

Nesta videoaula, eu mostro como é a estrutura básica de um documento HTML, explicando o significado e a utilidade de cada elemento. Explico também como é a estrutura dos arquivos do tipo CSS, dos seletores e blocos de propriedades.

Prof. Ricardo Maroquio


Aula 03 :: Editores de Código Repl.it e VSCode

Nesta videoaula, apresentarei os ambientes de desenvolvimento que utilizaremos ao longo do curso. O primeiro deles é o editor online Repl.it, que permite editar e executar códigos-fontes em mais de 50 linguagens de programação, incluindo HTML, CSS e Javascript. Em seguida, eu apresento o Visual Studio Code, um editor de código gratuito e o mais popular da atualidade.

Prof. Ricardo Maroquio


Aula 04 :: Elementos HTML Básicos para Exibição de Textos

Nesta videoaula, eu mostro os elementos HTML textuais básicos, incluindo elementos para títulos e subtítulos, parágrafos, listas, links, apresentação de código-fonte, entre outros. Além de mostrar como utilizar cada tipo de elemento, também falo a respeito da questão semântica de cada elemento.

Prof. Ricardo Maroquio


Aula 05 :: Elementos HTML para Exibição de Imagens

Nessa videoaula, eu mostro os elementos HTML usados para se exibir imagens, principalmente o elemento "img". Mostro também as diferenças entre imagens comuns e vetoriais, as formas de se mapear uma imagem em regiões clicáveis usando o elemento "map", e um recurso que permite carregar imagens específicas para determinadas resoluções de tela usando o elemento "picture".

Prof. Ricardo Maroquio


Aula 06 :: Elementos HTML para Exibição de Conteúdo Multimídia

Nesta videoaula, eu apresento os elementos HTML usados para se exibir conteúdos multimídia, a saber, áudio e vídeo. Mostro como exibir vídeos e áudios a partir de arquivos com formatos compatíveis com os navegadores atuais. Na parte dos vídeos, ensino como criar, editar e exibir legendas para os vídeos disponibilizados. Mostro também como incorporar vídeos e playlists do YouTube em uma página HTML.

Prof. Ricardo Maroquio


Aula 07 :: Criação de Imagens SVG usando HTML

Nessa videoaula, eu mostro os elementos HTML usados para se criar imagens vetoriais do tipo SVG, passando por todas as formas básicas e seus atributos e mostrando também como criar animações baseadas em elementos e atributos de elementos.

Prof. Ricardo Maroquio
