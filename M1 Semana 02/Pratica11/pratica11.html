1º Objetivo: Explorar o site de três bibliotecas CSS para conhecê-las e, ao fim, criar um documento HTML simples para vincular cada uma dessas bibliotecas de forma absoluta usando um repositório CDN. Basta digitar o nome da biblioteca seguido por “CDN” no Google para obter o link para a biblioteca em algum CDN. Ao fim dessa prática, ter 3 arquivos HTML, sendo que, cada arquivo HTML deve fazer referência a uma das 3 bibliotecas.

  1. Bootstrap (https://getbootstrap.com)
  2. Tailwind (https://tailwindcss.com)
  3. Foundation (https://get.foundation)

Resolução:

  1. Página com Bootstrap
  2. Página com Tailwind
